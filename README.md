  <h1>Selenium Cucumber Automation Testing Project</h1>

    This project is a complete automation testing framework developed using 
    <strong>Selenium</strong> and <strong>Cucumber (BDD)</strong>.

    <a href="https://automationexercise.com/">automationexercise.com</a>.

  </ul>

  <h2>Usage</h2>

  <p>To set up and run this project on your machine:</p>

  <ol>
    <li>Clone the repository:
      <pre><code>git clone YOUR_REPO_LINK_HERE</code></pre>
    </li>

    <li>Open the project in your preferred IDE (IntelliJ recommended).</li>

    <li>Install dependencies:
      <pre><code>mvn install</code></pre>
    </li>

    <li>Run the tests:
      <pre><code>mvn test</code></pre>
    </li>
  </ol>

  <h2>Reports</h2>

  <p>
    After execution, detailed test reports are generated automatically.  
    The report includes:
  </p>

  <ul>
    <li>Test status (Passed / Failed)</li>
    <li>Execution time</li>
    <li>Step-by-step scenario details</li>
    <li>Error logs for failed tests</li>
  </ul>
